# EstateView - Real Estate Appointment Scheduling and Payment System

**EstateView** is a modern **real estate appointment scheduling and property management system** built using the **Laravel (TALL Stack)**. The platform allows clients to browse properties, schedule visits, and make secure payments. It provides a streamlined experience for property inquiries, bookings, and management.

Visit the live app: [EstateView](https://estateview.online/)

Deployed on **Hostinger**.

---

## Features

### 👤 User Types
- **Clients**
  - Browse available properties
  - Schedule property viewings/appointments
  - Make payments for reservations or services
  - Track booking history and appointments

- **Admins**
  - Manage properties (create, update, delete listings)
  - Manage user accounts (clients)
  - View and manage appointments
  - Monitor payments and transactions

- **Agents / Staff (Optional Role)**
  - Manage assigned property listings
  - Handle appointment schedules
  - Assist clients with inquiries

---

## 🏡 Appointment Scheduling
- Clients can easily schedule **property visits**
- Admins can manage and confirm appointments
- Real-time scheduling system for better coordination

---

## 💳 Payment Integration (Stripe)
- Secure online payments using **Stripe**
- Clients can pay for:
  - Property reservations
  - Booking fees
  - Service charges
- Supports credit/debit card transactions

---

## 📊 Admin Dashboard
- Manage properties and listings
- Monitor all appointments
- Track payments and transaction history
- Manage users and system activity
